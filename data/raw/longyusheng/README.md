# 龙榆生《唐宋词格律》结构化数据

- 来源：longyusheng.org 站点开源仓库整理本 XML（仓库以 AGPL-3.0 发布其
  数字化工程；《唐宋词格律》原书为龙榆生（1902–1966）词学著作，通行权威）。
- `tangsong_cipu.xml`：原 XML 的 UTF-8 转存（内容未改动）。
- `cipu.jsonl`：由 `python3 -m hermes_poetry.corpus.cipu tangsong_cipu.xml`
  生成的结构化谱（153 调：调名/异名/类别/说明/定格与变格符号谱）。
  符号转写：－→○平 ＋→⊙可平可仄 │→●仄 ％→△平韵 ＊→▲仄韵；
  其余（＃可选增韵 、豆 ！～领格 ˇ衬字 ｛｝对偶 ［］叠韵 （）可省）保留原样。
- 用途：创作实验室与词牌定格的「词谱权威层」，与语料归纳定格并列展示、
  互为印证；输出恒标 `source_level: 词谱权威（龙榆生）`。
